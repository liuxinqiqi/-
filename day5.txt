类的定义

class  类名（object）：
	pass 

类名首字母通常大写

类的内容 ： 1.我们不建议写除了属性成员定义之外的语句
 	    2.内容主要包含 属性变量（实例变量）的定义，属性方法的定于




1. 类的定义，对象实例化   和对象相关内建函数  isinstance

2. 对象可修改原有属性，增加新的属性  和属性相关的内建函数  delattr(删除一个属性等价于del)  hasattr（判断是否有某个属性）   getattr（获取属性值）

3. 通过对象修改属性不会影响类中对属性的定义

4. 属性方法需要有self作为第一个参数，用作对象传入，在对象调用属性方法时self自动传参，不能人为传入

5. 类也可以调用属性变量和属性方法，但是类不可以调用对象的个性方法或属性

6. 类调用属性方法需要人为传入该类的一个对象  e.g.    
    
    class A():
	def test(self):
		pass

    obj = A()
    A.test(obj)

7. __init__函数用作对象的初始化，用于初始化区分出类的个性属性，在实例化对象的过程中会自动执行

8.一个类可以实例化多个对象，多个对象之间相互没有影响。对象都具有类定义的所有属性
